# App Router

Ce dossier est reserve a Next.js.

Il contient uniquement :

- `layout.tsx` : layout racine
- `page.tsx` : route `/`
- `api/` : futures routes backend exposees par Next.js

Le code de l'interface est dans `src/features`.
Le code partage est dans `src/shared`.
La logique serveur reutilisable est dans `src/server`.
