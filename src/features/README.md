# Features

Ce dossier contient les blocs fonctionnels de l'application.

```txt
src/features/
└─ portfolio/
   ├─ PortfolioApp.tsx
   ├─ theme.tsx
   ├─ index.ts
   └─ components/
```

Une feature regroupe ses composants, son etat local, ses types et ses helpers quand ils sont propres a cette partie du site.
